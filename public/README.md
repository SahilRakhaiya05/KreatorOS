# KreatorOS Public Static Assets

This directory contains public static assets for the KreatorOS application. Next.js serves all files in this directory from the root URL path (e.g. `/logo.png`, `/favicon.png`).

## Directory Structure
- `/logo.png` - The primary brand logo for KreatorOS.
- `/favicon.png` - The browser favicon.
- `/images/` - Public images and illustrations used throughout the app.

## Assets Generated
We have automatically generated a premium, futuristic brand identity logo for KreatorOS and saved it here as `logo.png` and `favicon.png`!
